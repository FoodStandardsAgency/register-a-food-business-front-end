cleanInactivePathAnswers(path, currentPage, movement) → {string}

Calculates the the next or previous page in the path, based on the active path (enabled pages), the current page, and the given 'movement'.
Parameters:
Name Type Description
path object The edited path object with the correct enabled and disabled pages
currentPage string The 'originator' page that the user has come from
movement number The requested movement, in number of pages, from the current page. For example 1 or -1
